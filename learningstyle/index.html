Learning Style
By Tripp Lyons




What is a learning style?
A learning style is a way that one learns best.
4 major learning styles are:
  • Visual
  • Auditory
  • Read-Write
  • Kinesthetic

What is my most preffered learning style?
I consider myself a Read-Write learner.
Some ways that I learn best are:
  • Typing
  • Reading
  • Writing
  • Taking Notes
  • Coding

What is my least preffered learning style?
My least preferred learning style is Kinesthetic. Some ways that I can adjust to be Kinesthetic are:
  • Exercising
  • Meditating By Focusing On My Body
  • Doing Sports
